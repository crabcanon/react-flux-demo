# react-flux-demo
* ES6 + webpack + Flux + React.
* Source from [React: Flux Architecture (ES6)](https://egghead.io/courses/react-flux-architecture-es6) by [egghead.io](https://egghead.io).

## Get started

* Clone the repo
* `npm install`
* `npm start`
* Open http://localhost:8080
